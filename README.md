# Tiny Steps｜小步一下

一个温柔的任务拆解网页。不是逼你高效，而是帮你开始。

## 当前版本

这是一个适合 GitHub Pages 的纯静态网页项目，不需要服务器或数据库。

功能包括：

- 输入一个当前卡住的任务
- 选择主要阻力和当前能量
- 自动生成 2 分钟启动、10 分钟推进、完成感收尾
- 支持更轻一点、更具体一点的重新拆解
- 使用浏览器本地存储保存最近记录

## 文件结构

- `index.html`：页面结构
- `style.css`：视觉设计
- `script.js`：任务拆解和本地保存逻辑

## GitHub Pages

可以在仓库 Settings → Pages 中开启。建议选择：

- Source: Deploy from a branch
- Branch: main
- Folder: /root

保存后等待 GitHub 生成网页链接即可。
